# qBittorrent UI Replacements

A list of alternative user interfaces for qBittorrent.

!!! note
    We've removed direct theme links because they don't always work across different qBittorrent versions.

## Recommended Options

We recommend using either [qui](https://github.com/autobrr/qui){:target="_blank" rel="noopener noreferrer"} or [VueTorrent](https://github.com/VueTorrent/VueTorrent){:target="_blank" rel="noopener noreferrer"}. Both offer much better performance than the default interface.

### qui

A modern alternative web interface for qBittorrent with multi-instance support. Built with Go and React.

### VueTorrent

A sleek web interface for qBittorrent built with Vue.js.

!!! tip "Why these are faster"
    Both options use the qBittorrent API directly, which makes them faster than the default GUI.
