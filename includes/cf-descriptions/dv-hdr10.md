This Custom Format will only match on `DV HDR10`

This format provides Dolby Vision _with_ HDR10 fallback.
You will find this mainly with:

- Remuxes
- Bluray encodes
- Hybrid WEB-DL (_Hybrid isn't always mentioned_)
