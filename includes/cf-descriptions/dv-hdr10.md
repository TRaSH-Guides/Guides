*This Custom Format will only match on* `DV HDR10`

This format provides Dolby Vision *with* HDR10 fallback.
You will find this mainly with:

- Remuxes
- Bluray encodes
- Hybrid WEB-DL (*Hybrid isn't always mentioned*)
