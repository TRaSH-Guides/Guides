### qui

[qui](https://github.com/autobrr/qui){:target="\_blank" rel="noopener noreferrer"} is a modern, multi-instance web UI for qBittorrent from the autobrr project, with a background automation engine that acts on torrents based on rules you define.

- Manage multiple qBittorrent instances from a single dashboard.
- Automations run on a periodic scan (roughly every 20 seconds) and can act on torrents by tag, category, speed/share limits, force recheck/reannounce, or run an external program — cross-seed matching itself is a separate qui feature, not an automation action.
- Developed and maintained by the autobrr project's own volunteer maintainers, so it's built with an autobrr + qBittorrent stack specifically in mind.
